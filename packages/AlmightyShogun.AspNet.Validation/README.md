# 🚀 `AlmightyShogun.AspNet.Validation`

A C# library that adds attribute and fluent request validation with standardized ASP.NET Core error responses.

## 📃 Prerequisites

- **[.NET](https://dotnet.microsoft.com/)**: >= 10

## 📦 Building

```sh
dotnet build packages/AlmightyShogun.AspNet.Validation/AlmightyShogun.AspNet.Validation.csproj
```
