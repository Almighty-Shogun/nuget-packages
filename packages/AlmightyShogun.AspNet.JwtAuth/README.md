# 🚀 `AlmightyShogun.AspNet.JwtAuth`

A simple C# library that adds JWT Auth with some extensions & utilities to ASP.NET.

## 📃 Prerequisites
- **[.NET](https://dotnet.microsoft.com/)**: >= 10

## 📦 Building

```sh
dotnet build packages/AlmightyShogun.AspNet.JwtAuth/AlmightyShogun.AspNet.JwtAuth.csproj
```
